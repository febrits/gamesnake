# GameSnake — Classic Snake Browser Game

The classic Snake game with modern UI and score tracking.

**Live:** https://gamesnake-two.vercel.app

## Features
- Classic snake gameplay
- Score tracking
- High score persistence
- Responsive controls

## Tech Stack
React 19 · TypeScript · Tailwind CSS v4 · Vite

## Getting Started
```bash
npm install
npm run dev
```
